# Global Coffee Sales Performance Dashboard (Excel)

## 📌 Project Overview
This repository contains an end-to-end Data Analytics and Business Intelligence solution built entirely in Microsoft Excel. The project transforms raw, disconnected transactional data across global markets into an interactive, executive-ready dashboard to drive strategic business decisions.

![Coffee Dashboard Preview](dashboard_preview.png)

---

## 🛠️ Tech Stack & Key Features
* **Tool:** Microsoft Excel
* **Data Architecture:** Built a unified relational data model across three source tables (`orders`, `customers`, `products`) utilizing advanced lookups (`XLOOKUP`, `INDEX/MATCH`).
* **Feature Engineering:** Automated data normalization, cleaned text anomalies, and engineered metrics for total line-item sales and human-readable product categorization.
* **Business Intelligence:** Implemented Pivot Tables, calculated fields, and interactive chart visualizations.
* **UI/UX Design:** Configured global **Slicers** and **Timeline controls** allowing stakeholders to dynamically filter entire subsets of data instantly.

---

## 🔍 Key Insights Discovered
* **Geographic Drivers:** Identified the primary international market driving gross revenue volume.
* **Customer Loyalty Value:** Segmented sales data to quantify the exact revenue uplift generated by Loyalty Card members compared to non-members.
* **Product Optimization:** Isolated top-performing coffee varieties and roasts, providing actionable data for inventory and supply chain budgeting.

## 📂 How to View the Project
Since GitHub does not render interactive Excel sheets directly in the browser, please follow these steps to explore the dashboard:
1. Download the `Coffee_Sales_Performance_Dashboard.xlsx` file from this repository.
2. Open the file in desktop Microsoft Excel.
3. Ensure macros/content are enabled if prompted, and click on the **Slicers** or **Timeline** on the `Dashboard` tab to test the interactivity!
